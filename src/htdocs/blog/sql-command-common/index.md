よく使う MySQL と PostgreSQL のコマンドの対応表です。

| 説明         | MySQL             | PostgreSQL      |
| ------------ | ----------------- | --------------- |
| DB 一覧      | `show databases;` | `\l`            |
| DB 選択      | `use DB_NAME`     | `\c DB_NAME`    |
| テーブル一覧 | `show TABLE_NAME` | `\d`            |
| テーブル構造 | `desc TABLE_NAME` | `\d TABLE_NAME` |
